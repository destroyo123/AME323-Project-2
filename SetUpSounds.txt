Hi Maren! Open a 'terminal' window by doing one of the following:
 - In Github Desktop app at the top, click "Repository" and click "Open in Terminal"
 - Or just open the "Terminal" app on your computer and run the following:

cd "your filepath to this folder"

Next, run these commands in that terminal to make the sound-playing scripts executable in MacOS.

chmod +x .githooks/post-merge
chmod +x scripts/play-sound.sh

This makes these scripts executable on MacOS...

Then also BOTH OF US need to run this to make GitHub run stuff when changes happen.

git config core.hooksPath .githooks
